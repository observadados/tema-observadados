== Description ==
O Tema ObservaDados é uma adaptação customizada do clássico tema "Twenty Sixteen" do WordPress. Ele mantém a estrutura robusta, acessível e responsiva (mobile-first) do original, mas foi modificado para atender às necessidades e identidade visual do projeto ObservaDados. 

Este tema é open-source e pode ser utilizado e adaptado livremente por terceiros sob os termos da licença GPL v2.0 ou superior. Sinta-se à vontade para utilizá-lo em seus próprios projetos!

* Baseado no clássico Twenty Sixteen
* Layout Responsivo e Mobile-first
* Otimizado para acessibilidade
* Customizações específicas de estilo e estrutura
* Licença GPL v2.0 ou posterior. :) Use para criar algo legal.

== Installation ==

1. Faça o download ou clone o repositório do tema.
2. Envie a pasta do tema para o diretório `/wp-content/themes/` da sua instalação WordPress (ou instale via Git Updater).
3. No painel de administração do WordPress, vá em Aparência -> Temas.
4. Clique no botão 'Ativar' no tema ObservaDados.
5. Navegue até Aparência > Personalizar no painel para ajustar as opções visuais.
